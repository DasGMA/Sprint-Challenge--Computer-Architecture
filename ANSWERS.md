* Convert `11001111` binary

    to hex: (1100)(1111) => (8+4+0+0)(8+4+2+1) => (12)(15) => C F

    to decimal: 128+64+8+4+2+1 => 207


* Convert `4C` hex

    to binary: (4)(C) => (0100)(1100) => 01001100

    to decimal: 64+8+4 = 4x16 + 12x1 => 76
